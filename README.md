# Horus

![Horus Eye](https://d30y9cdsu7xlg0.cloudfront.net/png/27083-200.png)

Rails Ready OpenWeather Maps gem, named after the Egyptian sky god

## Installation

Add this line to your `Gemfile`:

    gem 'horus'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install horus
